# Vue 3 + TypeScript + Vite

This template should help get you started developing with Vue 3 and TypeScript in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about the recommended Project Setup and IDE Support in the [Vue Docs TypeScript Guide](https://vuejs.org/guide/typescript/overview.html#project-setup).

//Admin//

The admin can manage all user data, including adding, editing, and deleting users.

The admin can also create appointments.

Admin Credentials:

Username: admin@nexware-global.com

Password: Admin@123

//User//

A user can view all appointment and manage only their own appointment data.

//Doctor//

A doctor can create their own doctor profile.

While creating a doctor entry, a reference code is required: REF = NEXWARE-GLOBAL.

A doctor can view their own appointments and patient details.

A doctor can modify only their own records; other doctors’ records are view-only.

//Patient//

A patient can create their own patient profile (patient ID).

A patient can book appointments.

A patient can modify only their own records; other patients’ records are view-only.

//Appointments//

Appointments for a patient with a doctor.

Patients can book appointments only with available doctors.
